Navigate inside Help

Navigate using the mouse

In the upperleft corner of the menu you can find two buttons.

Press the "Home" button to go back to the main page of this Help menu.

Use the "Back" button to go back one page of visited pages.

In the text you will find hyperlinks. When you hoover with the mouse pointer above boven such a hyperlink, the pointer will change to a hand. At that moment you can use the left mousebutton to go to the page it refers to.

Navigate using the keyboard

Using the [Home] button of your keyboard, will bring you back to the main page of the Help menu
Using the keyboard combination [Alt]+[Left Arrow] brings you back to previous visited page.
Using the keyboard combination [Alt]+[Right Arrow] brings you forward to a page in the sequence of visited pages.
Using the [Tab] button you can step over available hyperlinks in the text. After selection of a hyperlink, the [Enter] button will bring you to place where it refers to.
Using the [Esc] button you can close the Help menu.

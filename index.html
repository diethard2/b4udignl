Welcome to the homepage of the QGIS KLIC Viewer plugin

The QGIS KLIC Viewer plugin is developed and maintained by Diethard Jansen. If you have suggestions for improvement please contact me.. Any issues can be provided on the GIT-hub website.

How to install this plugin?

  • Install latest version of QGIS from download page at www.qgis.org.
  • Open the QGIS Desktop 3.x application.
  • In the menu Plugins select Manage and install plugins...
  • In the Search text field enter "KLIC"
  • In the result list you see KLIC Viewer
  • Press the button Install plugin
  • In the plugins toolbar, the icon of the KLIC viewer appears and you can start the KLIC Viewer for QGIS

For documentation on the plugin, please press the help button inside the menu of the plugin.
